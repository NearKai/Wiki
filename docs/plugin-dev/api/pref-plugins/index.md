# 前置插件提供的接口

由于 ToolDelta 是轻量的，所以它只提供了必要的与游戏交互的接口。

许多更高级的接口，需要各种前置插件提供。

比如，经典的聊天栏菜单的菜单注册接口，需要 `前置-聊天栏菜单` 插件提供。

## 前置插件的获取
在 ToolDelta 内置的插件市场内可以获取到文档里介绍的所有前置插件。
